IoT Data Pipeline en C++ avec tests en utilisant Clion 2025 .v3

Utilisation comme base de projet IoT :: 
Ce projet fournit un squelette orienté tests pour un pipeline IoT :

Remplacer les mocks par des implémentations réelles de capteurs et d’un client MQTT (par exemple Paho, Mosquitto) tout en conservant les interfaces ISensor, IMQTTClient, IStorage.

Étendre les règles de validation, les types d’alertes et les formats de sérialisation selon les besoins d’un système embarqué ou d’un backend IoT réel.
